🏺 How to Decipher the Script

1. Study the glyph sequence — Four symbols form a sentence in an unknown ancient language.
2. Read the clues carefully — Each describes WHERE a glyph was found and WHAT it depicts. Cross-reference them to deduce meanings.
3. Use logical deduction — If a glyph appears in a royal cartouche, it likely means KING. If it appears in a tomb scene showing death, it likely means DIED or REST. Process of elimination reveals the rest.
4. Assign meanings — Click buttons in the matrix. The game won't tell you what's correct — it only checks for logical consistency (no duplicate meanings).
5. Validate — When all 4 glyphs have meanings, type the full sentence. You'll know if you're right!
